Booting the Library

Rayfield is loaded with the following line:

local Rayfield = loadstring(game:HttpGet('https://sirius.menu/rayfield'))()

Configuration

Creating a window with configuration saving enabled:

local Window = Rayfield:CreateWindow({
  Name = "My UI",
  LoadingTitle = "Loading...",
  LoadingSubtitle = "Welcome!",
  ConfigurationSaving = {
    Enabled = true,
    FileName = "MyConfig",
  },
})

Configuration Saving

Enable configuration saving to keep user settings.

Loading Configuration

Load the configuration with:

Rayfield:LoadConfiguration()
